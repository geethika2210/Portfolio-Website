# 🚀 Portfolio-Website

A clean, minimal, and responsive portfolio template designed for software engineers. Built with **pure HTML and CSS**.

## ✨ Features
- 📱 Fully Responsive (Mobile + Desktop)
- 🎨 Modern UI with Tech Blue accents
- ⚡ Zero dependencies (Fast loading)
- 🛠️ Easy-to-edit commented code

## 🚀 How to Use
1. Copy `index.html` and `style.css` to your local folder.
2. Open `index.html` and look for `<!-- EDIT HERE -->` tags.
3. Open `index.html` in your browser to view!
